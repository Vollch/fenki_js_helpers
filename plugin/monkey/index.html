Userscripts for use in conjunction as browser extention with one of the following browser extension or by hand:

  • GreaseMonkey extension for Mozilla Firefox
  • TamperMonkey extension for Google Chrome
  • NinjaKit extension for Apple Safari or from here
  • ViolentMonkey Opera extension for both Presto and WebKit engines
  • To use userscripts with other incompatible browsers like Mobile Safari or Internet explorer, please consider at Bookmarklets: javascript:void(alert('Hi')); try press it
